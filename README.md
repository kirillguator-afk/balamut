# METRO CASH 🚇

P2P Платформа для игры в "Дурак" на внутреннюю валюту.

### Особенности:
- **Технология**: PeerJS (WebRTC) для прямой связи между игроками без сервера.
- **Интеграция**: Автоматическая публикация игровых офферов в Telegram канал.
- **Стиль**: Киберпанк/Метро эстетика с использованием Tailwind CSS.
- **Геймплей**: Классический "Дурак" (прототип 1х1).

### Как запустить:
1. Откройте `index.html`.
2. Введите ставку и нажмите "Опубликовать Вызов".
3. Ссылка на игру появится в Telegram канале.
4. Второй игрок переходит по ссылке и игра начинается автоматически.

### Конфигурация:
Токен бота и ID канала настроены в `tg_api.js`.
