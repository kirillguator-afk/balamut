# METRO CASH | P2P DURAK 🚇

Финальная сборка для тестирования.

### Технический стек:
- **PeerJS**: WebRTC соединение.
- **Telegram Bot API**: Шлюз для офферов.
- **Vanish UI**: Кастомный Metro-стиль.

### Как проводить тесты:
1. Откройте приложение на двух разных устройствах (или в разных браузерах).
2. На устройстве А нажмите "Опубликовать".
3. Скопируйте ссылку из Telegram канала или адресную строку (с хешем) и откройте на устройстве Б.
4. Игра начнется автоматически. Баланс 5000 ₽ выдается каждому игроку локально.
